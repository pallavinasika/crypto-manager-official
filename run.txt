Running the Python Backend & Streamlit Dashboard

Activate your virtual environment if you have one (or create one):
.venv\Scripts\activate

Install the required Python dependencies:
pip install -r requirements.txt

Initialize the database and generate some initial sample data:
python main.py setup

To run the Streamlit Dashboard:
python main.py dashboard

To run the FastAPI Backend/API server:
python main.py api

To run the background analysis engine:
python main.py analyze


Running the Modern React Web Frontend

1. Open a new terminal window.
2. Navigate to the web folder (CRITICAL):
   cd web

3. Install the Node.js project dependencies:
   npm install

4. Start the Vite development server:
   npm run dev
